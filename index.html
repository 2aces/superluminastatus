2017-08-26 15:19:51 UTC degraded performance Redes Sociais
Api Facebook instável: problemas para importação e exportação de conteúdo Há alguns dias nosso sistema está encontrando lentidão e instabilidade no acesso às APIs do Facebook e do Instagram, dificultando ou tornando possível a importação de conteúdos das redes sociais. Hoje, sábado, 26 de agosto, a API do Facebook e também o Facebook via navegador Chrome estava fora do ar para. Embora o site para desenvolvedores do Facebook não mencione estes problemas, empresas e usuários no mundo todo reportaram o problema. Infelizmente, não há muito o que nossa equipe possa fazer além de aguardar o retorno das operações normais das redes sociais. Após o retorno, faremos alguns testes e, após confirmar a normalização, atualizaremos este aviso e iremos disparar novamente o sistema de importação de conteúdo. Caso algum conteúdo não seja importado automaticamente, nossa equipe fará a importação manualmente. Agradecemos a compreensão.
